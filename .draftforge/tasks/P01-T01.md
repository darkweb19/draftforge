# P01-T01 — Implement local project initialization

Status: backlog

## Objective

Implement `draftforge init [directory]` so it creates a resumable, provider-neutral project without making a model call.

## Owned paths

- `src/commands/init.ts`
- `src/config/`
- `src/state/`
- `templates/`
- Relevant tests and documentation.

## Dependencies

- P00-T01

## Acceptance criteria

- Refuses to overwrite non-DraftForge files without explicit approval.
- Creates valid canonical state, `SESSION.md`, `AGENTS.md`, `CLAUDE.md`, phase files, and task directories.
- Re-running is idempotent or reports an actionable conflict.
- Works on Windows, macOS, and Linux path semantics.
- No provider, CLI login, or API key is required.
